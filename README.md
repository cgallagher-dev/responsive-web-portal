# Responsive Resource Portal

A full-stack Single Page Application (SPA) designed with a three-tier architecture. This project showcases asynchronous data fetching and RESTful API integration.

## Technical Highlights
* **RESTful API:** Developed a backend using the **Slim Framework (PHP)** to serve JSON responses to the client.
* **Asynchronous UX:** Implemented **AJAX** requests with jQuery to allow for real-time data updates without page reloads.
* **Database Integration:** Features a **MySQL** backend for persistent storage of resource metadata.

## Tech Stack
* **Front-end:** HTML5, CSS3, JavaScript (jQuery), Bootstrap
* **Back-end:** PHP (Slim Framework), Apache
* **Database:** MySQL
